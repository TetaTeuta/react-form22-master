React Form

React Form is a simple form that stores user input to API and renders the data from API. For more info about functionality see documentation.html Backend is build with Node.js and Express, data is stored to mongoDB and frontend is written in React.js. Test suits are made in Jest.
React Form consists of REST API and architected database built using JavaScript, Node.js, Express, and MongoDB.
